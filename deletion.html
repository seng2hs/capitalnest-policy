Delete Your Data

CapitalNest values user privacy and controls. Because our system architecture stores all tracking information locally on your device using an offline SQLite ecosystem, we do not host or hold your personal data on external web servers.

To permanently remove your application history:

  1. Open the CapitalNest application on your mobile device.
  2. Navigate to the Settings panel menu.
  3. Tap on the Reset Application option choice.
  4. Confirm the prompt choice by pressing Delete.

Alternatively, performing a standard Uninstall action via your native device settings manager will wipe all local databases instantly.

Notice: If automatic remote cloud sync modules are introduced in a future application release iteration, this dashboard path will be updated to include remote web portal single-click cloud deletion forms.
